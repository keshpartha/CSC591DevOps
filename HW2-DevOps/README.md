# HW2-DevOps
Test Generation

##Steps to execute

1. Clone the repository
2. Run npm install
3. Run node main.js
4. Run node_modules/.bin/istanbul cover test.js to get the coverage report.
5. Run open coverage/lcov-report/HW2-DevOps/(file-name).js.html to get a detailed report.
